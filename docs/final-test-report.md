## 6. Bug 修复记录

| 编号 | 问题 | 修复 |
| --- | --- | --- |
| B01 | 规则网关存在两个 Bean 导致后端启动失败 | 将 RuleEngine 网关设为主 Bean |
| B02 | 通知无法精确跳转业务目标 | 增加 target 字段并改造 NotificationHandler |
| B03 | 课程详情通知全量读取当前用户所有通知 | 新增课程通知过滤接口 |
| B04 | 学生“我的提交”前端循环课程/作业请求过多 | 新增后端聚合接口 |
| B05 | 疑似重复记录缺少教师/助教提醒 | DuplicateRecordService 生成重复提醒通知 |
| B06 | 学生提交聚合可能被传入 studentId 越权 | 接口不接收 studentId，只使用当前登录用户 |
